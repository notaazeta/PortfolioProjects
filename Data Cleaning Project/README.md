# Housing Data Cleaning Project (SQL)

This project demonstrates data cleaning and preparation using SQL on a housing dataset. The goal is to ensure the dataset is accurate, consistent, and ready for further analysis or visualization.

## Project Overview

- Cleaned raw housing data to handle missing, inconsistent, or incorrect entries.  
- Standardized data formats for dates, numeric fields, and categorical variables.  
- Prepared the dataset for exploratory analysis and dashboarding.  

## Tools & Technologies

- SQL (Microsoft SQL Server or any relational database)  
- Housing dataset (raw CSV)

## Key Tasks Performed

- Removed duplicates and irrelevant columns  
- Handled missing or null values  
- Converted data types to appropriate formats  
- Created cleaned tables/views for further analysis  

## Repository Contents

- SQL scripts for cleaning and transforming housing data  
- Views or tables ready for analysis  

## Skills Demonstrated

- Data cleaning and preprocessing  
- SQL queries and transformations  
- Preparing datasets for analysis and visualization
